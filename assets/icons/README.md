# Brand icons

White 96x96 PNG renders of glyphs from [Simple Icons](https://simpleicons.org)
(CC0 1.0 Universal). Shown on brand-colored backgrounds when the optional
"Brand icons" setting is enabled; services without a glyph fall back to a
brand-colored monogram.

Regenerate by rasterizing the corresponding `<slug>.svg` from the
`simple-icons` npm package at 96px with a white fill.

Some brands (Microsoft, Amazon, Xbox, Nintendo and others) are absent because
Simple Icons removed them at the trademark holder's request.
